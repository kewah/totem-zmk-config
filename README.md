# Totem ZMK Config

Personal ZMK firmware configuration for the [Totem](https://github.com/GEIGEIGEIST/TOTEM) keyboard.

Based on [GEIGEIGEIST/zmk-config-totem](https://github.com/GEIGEIGEIST/zmk-config-totem).

## Layers

### Layer Access

| Layer | Hold Key |
|-------|----------|
| NAV | RET |
| SYM | TAB or BSP |
| NUM | SPC |
| MEDIA | ESC |
| FUN | G-SPC |

## Legend

- `X/Y` = tap X, hold Y
- `HYP` = Hyper (Ctrl+Alt+Gui+Shift)
- `ML` = Mouseless (Hyper+Enter)
- `G-` = Gui/Cmd modifier
- `###` = layer activation key

### Morphs (shift behavior)

| Key | Tap | Shift |
|-----|-----|-------|
| `-/"` | `-` | `"` |
| `'/_` | `'` | `_` |
| `,/?` | `,` | `?` |
| `./!` | `.` | `!` |
| `/\` | `/` | `\` |
| `=>` | `=>` | `->` |

### Combos (NAV layer)

| Keys | Output |
|------|--------|
| H+A | Opt+Left (word left) |
| A+E | Opt+Right (word right) |
